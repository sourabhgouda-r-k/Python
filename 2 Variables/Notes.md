# **What is a Variable in Python?**

###### A **variable is like a container** that stores data.



###### Syntax: x = 10

###### Here:

###### &nbsp;  x = variable name (box)

###### &nbsp;  10 = value (thing inside the box)





###### Examples: 

###### name = "Sourabh"   (**name stores text**)

###### age = 21   (**age stores a number**)

###### 





#### **Rules for Naming Variables**



1. ##### **Allowed**

* ###### my\_name = "Sourabh"
* ###### age1 = 21
* ###### \_total = 100



##### 

##### **2. Not Allowed**

* ###### 1name = "Sourabh"   # cannot start with number
* ###### my-name = 10        # hyphen not allowed

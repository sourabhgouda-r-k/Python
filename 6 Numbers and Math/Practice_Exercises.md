# 🔢 Python Numbers \& Math - Practice Questions



## 🟢 EASY QUESTIONS (2)

### 1\. Basic Arithmetic

Write a program that:

* Takes two numbers as input
* Prints:

  * Addition
  * Subtraction
  * Multiplication
  * Division



### 2\. Floor Division \& Modulus

Write a program that:

* Takes two numbers
* Prints:

  * Floor division result (//)
  * Remainder (%)



## 🟡 MEDIUM QUESTIONS (2)

### 3\. Number Rounding

Write a program that:

* Takes a decimal number
* Prints:

  * Rounded value using round()
  * Ceiling value using math.ceil()
  * Floor value using math.floor()



### 4\. Type Conversion Checker

Write a program that:

* Takes a number as input
* Converts it into:

  * Integer
  * Float
* Prints both values and their types using type()



## 🔴 HARD QUESTIONS (2)

### 5\. Random Number Generator

Write a program that:

* Generates a random number between 1 and 50 using randint()
* Prints the number



### 6\. Number Validator

Write a program that:

* Takes a number
* Checks:

  * If it is integer using isinstance()
  * If it is whole number using is\_integer()

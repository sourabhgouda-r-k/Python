# Comments in code

* ###### **Comments are notes in code**

###### 

###### **Why comments are useful**

* ###### Understandable
* ###### Readable
* ###### Professional / TO explain code logic

1. ###### **Single-Line comment**

###### 2\. **Multi-Line comment(Multiple Lines).**

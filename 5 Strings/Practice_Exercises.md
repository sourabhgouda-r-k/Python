# 🧵 Python Strings - Practice Questions



## 🟢 EASY QUESTIONS (2)

### 1\. Length Finder

Write a program that:

* Takes a string input from the user
* Prints the length of the string using `len()`



### 2\. Upper \& Lower Case

Write a program that:

* Takes a word from the user
* Prints:

  * The word in lowercase
  * The word in uppercase



## 🟡 MEDIUM QUESTIONS (2)

### 3\. Replace Word

Write a program that:

* Takes a sentence from the user
* Replaces a specific word with another word using `replace()`

**Example:**
Input: I love Java  
Output: I love Python



### 4\. Split Words

Write a program that:

* Takes a sentence from the user
* Splits the sentence into words using `split()`
* Prints each word separately



## 🔴 HARD QUESTIONS (2)

### 5\. Username Validator

Write a program that:

* Takes a username as input
* Checks if it contains only alphabets using `isalpha()`

**Output:**

* "Valid Username" OR
* "Invalid Username"



### 6\. Word Search Program

Write a program that:

* Takes a sentence from the user
* Takes a word to search
* Checks if the word exists using `in`
* Prints the index using `find()`



## ⭐ BONUS QUESTIONS

### 7\. Remove Extra Spaces

Write a program that:

* Takes a string with extra spaces
* Removes spaces using `strip()`

**Example:**
Input: "   hello python   "  
Output: "hello python"



### 8\. Count Characters

Write a program that:

* Takes a string from the user
* Counts how many times a character appears using `count()`



### 9\. Check Start and End

Write a program that:

* Takes a string input
* Checks if it starts with a given word using `startswith()`
* Checks if it ends with a given word using `endswith()`

# What are **Data Types in Python**?

###### Data types tell what kind of value a variable is storing



#### **Core Data Types** in Python



##### **Integer (int)**

###### Used for whole numbers (no decimal)

###### a = 10

###### b = -5





##### **Float (float)**

###### Used for decimal numbers

###### x = 3.14

###### y = -2.5





##### **String (str)**

###### Used for text (words, sentences)

###### name = "Sourabh"

###### msg = 'Hello'





##### **Boolean (bool)**

###### Used for True or False values

###### is\_passed = True

###### is\_failed = False



##### **None**  

###### Means "no Value" it's used to show the absence of any data.

H = None

### What is **type()**?

###### **type()** is used to **check the data type of a variable** or value

###### 

###### syntax: type(variable)

###### 

###### Example:

###### x = 10

###### print(type(x))

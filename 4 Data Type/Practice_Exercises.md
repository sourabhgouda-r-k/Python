# 🐍 Python Data Types – Practice Questions



## 🟢 Easy Level

### 1\. Identify Data Types

Create variables:

* A number
* A decimal number
* A string
* A boolean

Print each variable along with its data type using `type()`.



### 2\. Simple Type Check

Write a program that:

* Takes input from user
* Prints the type of the input

👉 Observe what type is returned



## 🟡 Medium Level

### 3\. Type Conversion Practice

Write a program that:

* Takes a number as input
* Converts it into:

  * Integer
  * Float
  * String
* Prints all values with their types



### 4\. Boolean Check

Write a program that:

* Assigns values:

python
a = 10
b = 20



* Prints the result of:

  * `a > b`
  * `a < b`
  * `a == b`

👉 Also print their data types



## 🔴 Hard Level

### 5\. Mixed Data Types

Write a program that:

* Takes input for:

  * Name (string)
  * Age (int)
  * Marks (float)
* Prints all values with their data types



### 6\. Type Confusion (Important)

Write a program that:

python
a = "10"
b = 5



* Try adding them
* Observe the error
* Fix the error using type conversion
* Print the correct result

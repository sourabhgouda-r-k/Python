# 🐍 Python Input \& Type Conversion – Practice Questions

## 🟢 Easy Level

### 1\. Take User Name

Write a program that:

* Takes the user's name as input
* Prints a greeting message

**Example Output:**



Enter your name: Sourabh
Hello Sourabh





### 2\. Input and Print Number

Write a program that:

* Takes a number as input
* Prints the number

👉 Note: Do not convert the input (observe the behavior)



## 🟡 Medium Level

### 3\. Add Two Numbers

Write a program that:

* Takes two numbers as input
* Converts them to integers
* Prints their sum

**Example Output:**



Enter first number: 10
Enter second number: 20
Sum = 30





### 4\. Price Calculator

Write a program that:

* Takes product price as input
* Converts it to float
* Prints the final price

**Example Output:**



Enter price: 99.99
Final Price: 99.99





## 🔴 Hard Level

### 5\. Demonstrate String Behavior

Write a program that:

* Takes two numbers using `input()` (without conversion)
* Adds them directly
* Prints the result

👉 Observe what happens and explain why

**Example:**



Input: 10 and 20  
Output: 1020





### 6\. Mixed Data Input

Write a program that:

* Takes name (string)
* Takes age (integer)
* Takes height (float)
* Prints all values properly

**Example Output:**



Name: Sourabh
Age: 21
Height: 5.8

# **What is input() in Python?**

##### Input() is used to take **input from the user (keyboard)**

##### Syntax: input()



###### Example: 

###### &nbsp;name = input("Enter your name: ")

###### &nbsp;print("Hello", name)





##### **Note: "input() always takes data as STRING (text)"**





#### **Type Conversion**

##### Convert input into different types:

* ###### int()    # Integer
* ###### float()  # Decimal number
* ###### str()    # String





###### Example: 

###### a = int(input("Enter a number: "))

###### price = float(input("Enter price: "))

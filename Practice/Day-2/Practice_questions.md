# Python String Mastery Tasks (Day 2)

## ⏰ Deadline: 60–75 minutes

\---

## 🟢 Level 1 — Basic Control

1. Take a string and print it in reverse
Example: "python" → "nohtyp"
2. Count total characters (without directly using `len()` in output)
3. Convert entire string to uppercase without using `.upper()`
Hint: Use loops and ASCII values

\---

## 🟡 Level 2 — Logic Building

4. Count vowels and consonants separately
5. Check whether a string is a palindrome
Example: "madam" → Yes
6. Remove all spaces from a string

\---

## 🟠 Level 3 — Real Thinking

7. Count number of words in a sentence
8. Find the first non-repeating character
Example: "aabbcde" → c
9. Replace all vowels with `\*`
Example: "hello" → "h*ll*"

\---

## 🔴 Level 4 — Challenge

10. Check if two strings are anagrams
Example: "listen" and "silent" → Yes
